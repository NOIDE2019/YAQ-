# Modelo de negocio (MVP → Fases)

## MVP (Fase 1)
- Comisión por cierre (afiliado/lead/venta) o fee por partner
- En Tigre: empezar con pocos partners y controlar calidad

## Fase 2
- Suscripción Premium (mejor curaduría, acceso anticipado, garantía mejorada)
- Swap 1:1 dentro de subcat + rango

## Fase 3
- Drops/limitadas
- Subasta/permuta con reglas duras (si aplica)

## Fase 4
- Colección utilitaria (no NFT): beneficios futuros definidos (acceso, upgrades, perks)
